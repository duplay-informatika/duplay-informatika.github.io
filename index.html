Arsitektur CPU

Komponen Utama CPU:

Control Unit (CU)

Mengatur dan mengendalikan semua operasi dalam CPU. CU membaca instruksi dari memori, menerjemahkannya, dan mengatur eksekusi oleh komponen lain.

Arithmetic Logic Unit (ALU)

Melakukan operasi aritmetika (+, -, *, /) dan operasi logika (AND, OR, NOT). ALU adalah "otak" matematika dari komputer.

Register

Penyimpanan sementara yang sangat cepat di dalam CPU. Register menyimpan data dan instruksi yang sedang diproses.

Memori Utama (RAM)

Menyimpan data dan instruksi yang sedang digunakan CPU. Lebih cepat dari memori sekunder tetapi volatile (hilang saat daya mati).

Instruksi Dasar CPU

Jenis Instruksi:

  • LOAD - Memindahkan data dari memori ke register
  • STORE - Memindahkan data dari register ke memori
  • ADD/SUB - Menambah/mengurangi nilai di register
  • MULT/DIV - Mengalikan/membagi nilai di register
  • JUMP - Melompat ke instruksi tertentu

Siklus Instruksi:

  1. Fetch - Mengambil instruksi dari memori
  2. Decode - Menerjemahkan instruksi
  3. Execute - Menjalankan instruksi
  4. Store - Menyimpan hasil
Simulasi Eksekusi Program CPU

Program Sederhana:

1 LOAD R1, [M1] ; Memuat nilai 10 dari M1 ke R1
2 LOAD R2, [M2] ; Memuat nilai 5 dari M2 ke R2
3 ADD R1, R1, R2 ; Menambahkan R1 dan R2, hasil di R1
4 STORE [M3], R1 ; Menyimpan hasil dari R1 ke M3
